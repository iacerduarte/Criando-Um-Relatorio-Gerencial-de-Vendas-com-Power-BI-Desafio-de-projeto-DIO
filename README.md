# Criando-Um-Relatorio-Gerencial-de-Vendas-com-Power-BI-Desafio-de-projeto-DIO

## 📊 Financial Report UX Optimization | Power BI Project<b>
👤 Sobre o Projeto<b>

Este projeto demonstra minha capacidade de transformar um relatório financeiro em um dashboard orientado à experiência do usuário (UX), combinando análise de dados, design de informação e interatividade no Power BI.<b>

O foco principal foi melhorar a usabilidade, interpretação dos dados e tomada de decisão, aplicando boas práticas de visualização e navegação.<b>

## 🎯 Objetivo<b>

Reestruturar um relatório financeiro existente para:<b>

Melhorar a leitura e interpretação dos dados
Criar uma experiência de navegação intuitiva
Aplicar princípios de UX/UI em dashboards analíticos
Aumentar o valor analítico das informações apresentadas<b>
## 🧠 Principais Competências Demonstradas<b>
Data Visualization (Power BI)
UX aplicado a dados (Data UX)
Design de dashboards
Storytelling com dados
Organização e hierarquia da informação
Interatividade e navegação guiada
Pensamento analítico orientado a negócio<b>
## 🛠️ Tecnologias e Ferramentas<b>
Power BI Desktop
DAX (básico/intermediário)
Modelagem de dados (estrutura analítica)
Princípios de UX/UI<b>
## 🚀 Soluções Implementadas<b>
🔹 Navegação Inteligente
Criação de botões interativos
Menus de navegação em todas as páginas
Destaque visual para estado ativo/selecionado
🔹 Melhoria de Layout e UX<b>
Reorganização estratégica dos elementos
Aplicação de contraste visual para priorização de dados
Uso de hierarquia visual clara<b>
🔹 Otimização da Experiência do Usuário<b>
Redução de ruído visual
Fluxo de navegação mais intuitivo
Melhor segmentação das informações<b>
📊 Estrutura do Dashboard<b>

O relatório contém 3 páginas principais:<b>

Overview Financeiro
Visão geral com KPIs e indicadores principais
Análise Detalhada
Exploração mais profunda dos dados financeiros
Insights Estratégicos<b>
Destaques e apoio à tomada de decisão<b>
📈 Resultados e Impacto
📉 Redução da complexidade visual
⚡ Melhoria na velocidade de interpretação dos dados
🧭 Navegação mais intuitiva e orientada
📊 Aumento da clareza dos insights apresentados
